# Claude Codeで企業サイト構築：AI開発パートナーとの協働実践記録

**公開日**: 2025年1月18日  
**著者**: ルーア・ブランカ  
**カテゴリ**: AI活用事例  
**タグ**: Claude Code, Next.js, AI開発, 実践記録

## はじめに

LUA BLANCA CONNECTのサイト構築において、Anthropic社のClaude Codeを活用した開発を実践しています。本記事では、実際の開発プロセスで得られた知見、つまずいたポイント、そして従来の開発手法との比較を詳細に記録します。

## プロジェクト概要

### 開発対象
- **サイト名**: LUA BLANCA CONNECT
- **技術スタック**: Next.js 15, TypeScript, Tailwind CSS, Sanity CMS
- **開発期間**: 2025年1月17日〜継続中
- **開発スコープ**: コーポレートサイト（問い合わせフォーム、ブログ機能含む）

### 開発体制
- **開発者**: 1名（筆者）
- **AIパートナー**: Claude Code
- **CMSセットアップ**: 外部委託済み

## 実際の開発プロセス記録

### 1. Formspree問い合わせフォーム実装（約2時間）

**課題**: 既存フォームをFormspreeに連携し、自動返信機能を追加

**AI活用のメリット**:
- フォーム送信ロジックの実装が迅速
- エラーハンドリングの網羅的実装
- レスポンシブデザインの自動最適化

**つまずいたポイント**:
- Formspreeの自動返信機能設定
  - コード側での`_autoresponse`実装を試行
  - 最終的に管理画面での設定が必要と判明

**従来開発との比較**:
- 従来: 約4-6時間（調査、実装、テスト含む）
- AI活用: 約2時間（50-70%短縮）

### 2. サービス選択肢の更新（約30分）

**課題**: 問い合わせフォームのサービス選択肢を最新に更新

**実際の変更**:
```typescript
// 変更前: 汎用的な選択肢
// 変更後: 具体的なサービス名
'AI活用コンサルティング',
'AIマーケティング戦略', 
'AI活用地方生産者支援',
'生成AI',
'コーディングAI活用支援',
'AIプロジェクト管理支援',
'AI時代のコンテンツ戦略',
'その他'
```

**AI活用の効果**:
- 瞬時にコード修正提案
- エラーなく一発実装
- 従来: 約1時間 → AI活用: 約30分

### 3. ブログ→更新情報への構造変更（約3時間）

**課題**: `/blog`から`/updates`への大規模リネーム作業

**実装内容**:
- ディレクトリ構造変更
- 全URLパスの更新
- ナビゲーション修正
- 内部リンク一括変更

**AI活用の強み**:
- 影響範囲の網羅的把握
- 一括変更作業の自動化
- 見落としがちな関連ファイルの特定

**課題と解決**:
- 複数箇所の同一文字列変更で`replace_all`パラメータ活用
- ファイル間の依存関係をAIが自動認識

### 4. レイアウト設計・実装（約4時間）

**実装フェーズ**:
1. サイドバー付きレイアウト（案1）
2. フル幅レイアウト（案2）  
3. 最終統合版（案1ベース + 改良）

**AI開発の特徴**:
- 複数案の迅速なプロトタイピング
- デザイン一貫性の自動維持
- レスポンシブ対応の網羅的実装

**デザイン仕様**:
```css
/* ブランドカラー統一 */
primary: #2D5A5A
secondary: #5BBCB6  
accent: #A8D5D0
highlight: #F6E96B
```

## AI開発のメリット・デメリット

### メリット

1. **開発速度の劇的向上**
   - 実装アイデアの即座な具現化
   - ボイラープレートコードの自動生成

2. **品質の向上**
   - ベストプラクティスの自動適用
   - エラーハンドリングの網羅性

3. **学習効果**
   - 実装を見ながら技術習得
   - 新しいアプローチの発見

### デメリット・制約

1. **外部サービス連携の限界**
   - Formspree設定など、API仕様の詳細把握が必要
   - サードパーティ管理画面は別途対応が必要

2. **デバッグ時の複雑さ**
   - AI提案コードの動作原理理解が必要
   - トラブルシューティングスキルは依然重要

3. **創造的判断の必要性**
   - UIデザインの最終判断は人間が必要
   - ビジネス要件の解釈は人間の領域

Claude Codeを活用した開発により、**約70%の工数削減**を実現しています。特に以下の分野で効果が顕著でした：
